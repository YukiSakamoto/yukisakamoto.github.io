    計算化学関係のツール・リンク

    計算プログラム

    • Quantum ESPRESSO: 平面波基底を用いた第一原理計算プログラム
    • Gaussian: Gauss関数の局在基底を用いた量子化学計算プログラム
    • ORCA: 局在基底を用いた量子化学計算プログラム

    Quantum ESPRESSO関係

    • Input Description: QE公式のマニュアル。頻繁に使うものは、pw.x, ph.x, dynmax.x, pp.x, projwfc.x
    • 擬ポテンシャル
    • Quantum ESPRESSO tutorial: QEでのSCF計算・フォノン他様々な計算の実行方法の解説。
    • Quantum ESPRESSO入力ファイル作成手順: クロスアビリティ社の方による解説記事。特に擬ポテンシャルの選択方法など参考になります。

    Gaussian関係

    • Gaussian 16 Users Reference: リファレンス(公式)
    • List of Gaussian Keywords : キーワードリスト(公式)

    計算手法など

    • Basis Set Exchange: 原子局在基底関数のデータベース。そのままGaussianの入力なども作成可能。

    可視化ツール

    • VESTA: 結晶構造の可視化は大体これを使っています。
    • XCrySDen: 描画の美しさはVESTAの方が上ですが、QEのinput/outputを直接読めるので便利。構造最適化(relax)の過程も描画可能です。
    • AFLOW: 1st BZ の作成に主に利用しています。
    • Materials Cloud Tools: 主にInteractive phonon visualizerを利用しています。

    結晶構造・データベース

    • CCDC
    • COD (Crystallography Open Database) : CCDCよりも動作がかなり軽快。
    • Springer Materials
    • Materials Project : 構造・物性の計算結果のデータベース

    • 結晶構造ギャラリー: 産総研が公開している結晶構造ライブラリ。cifが入手可能。

    • CCCBDB: 量子化学計算と実験値（スペクトル）を対応させるためのデータベース。振動のスケールファクターなどはたまに見ます。
